## Artifex Changelog

### Additions
- New API changes: Specialties are now data-driven and can be added to any weapons, though proper functionality and compatibility is not guaranteed.
- New API changes: You can now register your Creative Tabs to be animated.
- Combo stacking HUD texture is changed.

### Adjustments
- [General ~] Adjustments are made for the combo stacking timer to scale properly with the weapon's melee attack speed (Proper functionality and compatibility is not guaranteed with throwable and ranged weapons).
- [Weapon ~] Katana: Adjusted BetterCombat damage multiplier distributions for different attacks.
- [Weapon ↓] Battle Axe: Reduced damage scaling based on item tier.
- [Weapon ↓] Sickle: Reduced damage scaling based on item tier.
- [Specialty ↓] Iaijutsu: Timer now resets when the holder is hurt and timer heavily increases based off on specialty level.
- [Specialty ↑] Iaijutsu: Timer speed now triples when player is standing still or moving slowly.
