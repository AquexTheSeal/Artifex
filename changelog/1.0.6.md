## Artifex Changelog

### Additions
- Added new specialty: Iaijutsu
- Added new weapon type: Scimitar
- Added new weapon type: Rapier
- Added configuration to change the render offset of combo overlays and throwable overlays
- Changed execute specialty particle animation

### Adjustments
- [Weapon ~] Katana specialties: Added Iaijutsu II, Added Two-Handed I, removed Sweeping II/Finesse II
- [Weapon ~] Odachi specialties: Added Iaijutsu I, removed Sweeping I/Finesse I
- [Weapon ↓] Odachi reach bonus: 1.2 -> 1.0
- [Weapon ↓] Katana damage bonus: 2.5 -> 2.0
- [Weapon ↓] Katana attack speed bonus: 1.75 -> 1.5
- [Weapon ↓] Dagger maximum drawing time: 0.25s -> 0.5s
- [Weapon ↓] Spear maximum drawing time: 0.5s -> 0.75s
- [Weapon ↓] Battle Axe attack speed bonus: 0.8 -> 0.75
