## Artifex Changelog

### Additions
- Added a throw time visualizer in the crosshair for all throwable weapons.
- Thrown item projectiles no longer play the arrow sound when landing on the ground.

### Adjustments
- [General ~] Throwable damage tier scaling: 1.2 -> 1.1
- [General ~] Throwable damage now scales with throw time
- [Weapon ↓] Spear throwing base damage: 5.0 -> 4.5
- [Weapon ↑] Javelin throwing base damage: 4.5 -> 6.0
- [Weapon ↑] Javelin throwing base velocity: 3.0 -> 5.0
