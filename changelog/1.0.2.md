## Artifex Changelog

### Fixes
- Fixed crash with collective
- Attack speed stacking for weapons with the Finesse specialty are much smoother now
- Ranged (crossbow and bow) weapons no longer benefit from specialties' melee abilities

### Adjustments
- [Weapon ↓] Knuckles base damage bonus: 1.5 -> 1.25
- [Weapon ↓] Shortsword specialties: Finesse II -> Finesse I
- [Weapon ↓] Halberd specialties: Crippling II -> Crippling I
- [Weapon ↓] Halberd attack speed: 1.2 -> 1.0
- [Weapon ↓] Glaive attack speed: 1.0 -> 0.9
- [Weapon ↓] War Door disable cooldown multiplier: 0.0 -> 0.3
- [Specialty ↓] Armor Piercer damage scaling: [0.2 + (level * 0.15)] -> [0.15 + (level * 0.15)]
